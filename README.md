One paper per day
=================

A tiny script downloading the latest <= 20 papers that are offered on Twitter
courtesy [@onepaperperday](https://twitter.com/onepaperperday).

Usage
=====

```
$ onepaperperday 1
2014-03-27 21:09:20 http://dld.bz/dm4WP 129902.129906.pdf
```
